### LayeAwards Management

Internal project management software for LayeAwards.

Live: https://management.layeaward.com

#### Features

- Task tracker with department, status, and deadline per task
- Sponsor pipeline with proposal status and pledged amounts
- Live voter, nominee, and revenue figures from the voting site
- Realtime sync across the team
- Password protected deletes

#### Tech stack

- React
- TypeScript
- Vite
- Installable PWA
